# Blink exercise

Write a jQuery plugin that makes an element act like a `<blink>` tag.  It should work for any arbitrary speed.

```javascript
// show/hide every 1000ms
jQuery('.myDiv').blink(1000);
// twice as fast
jQuery('.otherDiv').blink(500);
```
